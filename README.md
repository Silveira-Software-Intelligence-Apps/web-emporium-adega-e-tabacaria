# Emporium Adega E Tabacaria

Site profissional gerado por **Lead Machine** — Next.js 14 + Tailwind CSS.

## Nicho
tabacaria — Barueri

## Stack
- Next.js 14 (App Router)
- Tailwind CSS 3
- React 18
- Google Fonts (Inter)

## Rodar local
```bash
npm install
npm run dev
```

## Deploy
```bash
npm run build
```
